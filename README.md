# Airbnb Clone 🏠 (In Progress)

A full-stack Airbnb clone built using Node.js, Express, MongoDB, EJS, and Bootstrap.

## Features So Far
- Listing creation
- Image display
- Review system
- Authentication

## Tech Stack
- Node.js
- Express
- MongoDB + Mongoose
- EJS
- Bootstrap 

## Status
🚧 Still in development – working on final features like validation, user permissions, and deployment.

## Setup
1. Clone the repo
2. Run `npm install`
3. Add your `.env` with your database URL and session secret
4. Run `node app.js` or `nodemon app.js`
